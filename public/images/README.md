# Image Asset Guide

Keep static images grouped by the page or module that owns them.

- `common/`: shared logo, badge, arrows, ticks, and page hero backgrounds.
- `home/`: homepage-only assets, grouped by section.
- `download/`: download page platform cards, mobile backgrounds, and reason icons.
- `login/`: login page background and auth mode icons.
- `pricing/`: rendered price images used by membership cards.
- `profile/`: profile and membership images.
- `sdk/`: SDK page feature icons and demo cards.
- `platforms/`: live platform icons used by the homepage carousel.

Naming rules:

- Use lowercase English words.
- Use hyphen-separated names, for example `mobile-android-bg.png`.
- Avoid pinyin, numeric-only names, and vague names like `icon1.png`.
- Put new assets in the nearest module folder instead of the root `images/` folder.
